# O(log n) Exponentiation by Squaring

## Amaç
Bu projede üs alma işlemi klasik O(n) yöntem yerine
Exponentiation by Squaring algoritması kullanılarak
O(log n) zaman karmaşıklığında hesaplanmıştır.

## Kullanılan Yöntem
Üs değeri her adımda ikiye bölünmektedir.
Bu nedenle işlem sayısı log(n) seviyesinde artar.

## Zaman Karmaşıklığı
O(log n)

## Örnek
power(2, 10) = 1024
